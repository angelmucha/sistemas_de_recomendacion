# Sistema de Recomendación para Aprendizaje Personalizado de Matemáticas

Implementación de un sistema de recomendación basado en Machine Learning para personalizar el aprendizaje de matemáticas en educación primaria peruana.

## Módulo 1: Preprocesamiento de Datos

### Instalación
```bash
pip install -r requirements.txt
```

### Uso
```bash
python scraper_multioperacion.py
python metrics_collector.py
```

### Resultados

- 425 problemas matemáticos extraídos
- 4 tipos de operaciones (suma, resta, multiplicación, división)
- 96.94% de problemas únicos
